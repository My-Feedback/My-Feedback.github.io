About MyFeedback

The problem.

A big issue advertisers and large content creators face is that they can’t connect with their large audience very easily. When they create a post or make a video that gets thousands of comments it can seem almost impossible to read them all and understand what your audience wants.

Our Solution

MyFeedback will help these people understand their audience. MyFeedback will be a web app were the user can upload the URL of their post and the program will do the rest. The program will read all of the post’s comments and put them through a sentiment analysis program that will return key factors about what the audience is saying.

The Pathway to Sucess

Data gathering process consists of gathering large amounts of data for the sentiment analysis program to read. The way our team will gather the data will be using web scrapers to gather the comments from the post the user selects. Our plan is to get one website scraper working first (YouTube) before expanding to other websites.

Sentiment Analysis is the main ‘brain’ of MyFeedback and we plan on using Microsoft’s sentiment analysis API to read all the data we submit. The API will return the important data we need and send it to our website for the user’s to easily read.

User Interaction will be one of the most important parts due to one of the main idea of MyFeedback is to help the user’s understand their audience. To help them understand their audience they need to be able to read the data we gather and analyze easily and efficiently. To do this we plan on using material design and bootstraps to help us design a beautiful and readable website.
